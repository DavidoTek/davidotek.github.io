ProtonUp-Qt

Install and manage Proton-GE installations for Steam with this graphical user interface.
Based on AUNaseef's ProtonUp, made with Python 3 and Qt 6.

Download ProtonUp-Qt

  1. Download the ProtonUp-Qt AppImage.
  2. Mark the AppImage as executable (either using your file manager or using the Terminal: chmod +x ProtonUp-Qt*.AppImage)
  3. Double-click the AppImage to run ProtonUp-Qt
  4. (Optional: Integrate the AppImage into your desktop. More information about AppImages here)
